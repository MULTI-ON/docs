# List Running Sessions

List the currently running sessions with the MultiOn Agent. This is crucial for managing and monitoring ongoing interactions with the API.

- **Purpose**: Listing running sessions provides an overview of all active interactions, allowing for better management and oversight.
- **Use Cases**: Useful for tracking session activity, diagnosing issues, or understanding user interactions at any given time.

## Listing Running Sessions

Retrieve a list of all active sessions using the following sample code:

<Tabs items={['Python', 'JS', 'Curl']}>
  <Tab>
  ```python copy
    multion.list_sessions()
  ```
  </Tab>
  <Tab>
  ``` js copy 
    await multion.list_sessions()
  ```

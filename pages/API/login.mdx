## Authorization

By default, the Multion API uses OAuth2 for secure authentication. This method ensures the safety and privacy of your data. Follow the standard OAuth2 flow to authenticate your application.

<Tabs defaultValue="python" items={["Python", "JS", "Curl"]}>
  <Tab value="python" label="Python">
    ```python copy 
    import multion 
    multion.login()
    ```
  </Tab>
  <Tab value="js" label="JavaScript">
    ``` js copy 
    import multion from 'multion'; 
    await multion.login(); 
    ```

## Optional: Generating API Keys

If you prefer using API keys instead of OAuth2, you can generate them with the `use_api=True` option in your login method.

<Tabs defaultValue="python" items={['Python', 'JS', 'Curl']}>
  <Tab value="python" label="Python">
  ```python copy
  # Generate API Key using Python
   multion.login(use_api=True)
````
</Tab>
<Tab value="js" label="JavaScript">
``` js copy
// Generate API Key using JavaScript
await multion.login({use_api: true})
````

  </Tab>
  <Tab value="curl" label="Curl">
  ```bash copy
  # Visit the following URL to generate an API key: https://app.multion.ai/api-tokens
  # Save the API key in the MULTION_API_KEY environment variable.
  ```
  </Tab>
</Tabs>

- Your API key or access token will be used automatically for all subsequent API requests.

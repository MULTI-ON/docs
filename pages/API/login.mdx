## Authorization
By default, the Multion API uses OAuth2 for secure authentication. This method ensures the safety and privacy of your data. Follow the standard OAuth2 flow to authenticate your application.

<Tabs items={['Python', 'JS', 'Curl']}>

  <Tab>

  ```python copy

  ```
  </Tab>
  <Tab>

## Optional: Generating API Keys
If you prefer using API keys instead of OAuth2, you can generate them with the use_api=True argument in your login method.

<Tabs items={['Python', 'JS']}>
<Tab>

python
Copy code
# Generate API Key using Python
multion.login(use_api=True)
  </Tab>
  <Tab>
javascript
Copy code
// Generate API Key using JavaScript
await multion.login({use_api: true})
  </Tab>
</Tabs>

- The API key will be automatically stored in your system and used for subsequent API requests.

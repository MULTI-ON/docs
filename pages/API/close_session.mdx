# Close a Session

Close an existing session with the MultiOn Agent. This is useful for conclude an interaction securely and efficiently.

## Understanding Session Closure

- **Purpose**: Closing a session terminates the current interaction, ensuring that no further processing or data exchange occurs in that session.
- **Best Practices**: It's recommended to close sessions when they are no longer needed to free up resources and maintain optimal performance.

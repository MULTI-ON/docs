## Step Session

To step a session, you'll need to provide the session ID (sessionId) and the new input parameters.

### Required Parameters

- **sessionId**: The unique identifier of the session you wish to update.
- **input**: The new or additional user instruction for the session.
- **url**: The URL relevant to the updated request.

<Tabs items={['Python', 'JS', 'Curl']}>

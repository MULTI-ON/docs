## Step Session

To step a session, you'll need to provide the session ID (sessionId) and the new input parameters.

### Required Parameters

- **sessionId**: The unique identifier of the session you wish to update.
- **input**: The new or additional user instruction for the session.
- **url**: The URL relevant to the updated request.

### Optional Parameters

- **stream**: Whether to stream the response or all at once.

<Tabs items={['Python', 'JS', 'Curl']}>
  <Tab>
```python
data = {
  "input": "what is the weather today",
  "url": "https://www.google.com"
}
response = multion.step_session(sessionId, data)
```

### Streaming API

<Tabs items={['Python', 'JS', 'Curl']}>
  <Tab>
```python
data = {
  "input": "what is the weather today",
  "url": "https://www.google.com",
  "stream": True
}
response = multion.step_session(sessionId, data)
for chunk in response:
  if chunk:
      print(chunk)
```
